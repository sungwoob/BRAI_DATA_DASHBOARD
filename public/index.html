BRAI Data Dashboard

API 사양서(v1.0)에 기반한 데이터 시각화 도구입니다. 서버 환경 변수로 실제 API 연동 여부를 제어합니다.

데이터세트 목록

GET /api/datasets (서버: /api/dataset)

데이터세트 상세

POST /api/dataset/:id (로컬 GET /api/datasets/:id)

계통 상세 & 표현형 시각화

POST /api/strains/:id (로컬 GET /api/strains/:id)

기본 API URL: . 실환경에서는 API_BASE_URL과 USE_MOCK 환경변수를 설정하세요.
